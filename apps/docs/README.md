# Marble Documentation

Documentation for Marble and its API built with [Mintlify](https://mintlify.com).

## Development

From the monorepo root:

```bash
pnpm docs:dev
```

Or from this directory with the [Mintlify CLI](https://www.npmjs.com/package/mint):

```bash
pnpm add -g mint
mint dev
```
